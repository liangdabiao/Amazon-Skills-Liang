# 可复用 Listing 模板 — Toy Story 角色 IP 类公仔

> 基于竞品分析提取的最佳实践模板，适用于 IP 授权角色公仔品类

## 模板适用范围

- IP 授权角色公仔（Disney/Pixar、漫威、DC、任天堂等）
- 带发声功能的互动玩具
- 12-20 英寸大型公仔
- 收藏级/礼品级玩具

---

## 模板一：标题模板

```
[Brand] [IP/电影名] [角色名] [核心功能] [关键特征 1], [关键特征 2], [差异化特征] — [目标人群]
```

**Mattel Roundup Fun (B09CGM4PVM)**：
> Mattel Disney Pixar 玩具总动员会说话的 [角色名] 动作人物和配件, Roundup Fun [尺寸] 英寸收藏版, [数量] 多种声音和短语

**原始**：Disney 玩具总动员 16 英寸会说话的木制娃娃
**建议**：Disney 玩具总动员 16 英寸拉线会说话 Woody 公仔, 10 个经典电影短语, 可拆卸牛仔帽, 柔软身体适合拥抱 — 完美礼物

## 本品类 Listing 要素清单

### 必须包含的要素
- [ ] IP 名称（Toy Story / Disney Pixar）
- [ ] 角色名（Woody / Buzz / Jessie）
- [ ] 核心功能（Talking / Interactive / Pull-string）
- [ ] 短语数量（10 phrases / 30+ sounds）
- [ ] 尺寸（16 inch / 12 inch）
- [ ] 材质（Soft body / Plastic head / Fabric）
- [ ] 特殊部件（Removable hat / Pop-out wings / Light-up laser）

### 建议包含的要素
- [ ] 电影经典台词引用
- [ ] 与其他角色的互动特性
- [ ] 收藏定位（Collector / Gift / Play）
- [ ] 年龄范围（Ages 3+）
- [ ] 包装描述
- [ ] 安全认证
