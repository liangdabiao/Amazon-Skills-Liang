## 一、标题模板提取

### 1.1 当前标题结构（基础版）

```
[IP角色名] + [尺寸] + [功能类型] + [配件说明] + [年龄范围] + [授权信息]
```

实际拆解：
- Sonic the Hedgehog → IP角色名
- 4 英寸 → 尺寸
- 声波动作人偶 → 功能类型
- 带蓝色检查点配件 → 配件说明
- 适合 3 岁以上儿童 → 年龄范围
- (Sega 官方授权) → 授权信息

### 1.2 优化标题模板

**模板A：IP授权动作公仔（英文，美国市场）**

```
[品牌名] [IP角色名] [尺寸] [版本/变体] Action Figure with [配件描述], [授权信息], [关键词] Toy Gift for Kids Ages [年龄]
```

**示例（Sonic优化版）：**
> JAKKS Pacific Sonic the Hedgehog 4-Inch Modern Sonic Action Figure with Blue Checkpoint Accessory, Officially Licensed by SEGA, Articulated Collectible Toy Gift for Kids Ages 3+

**模板B：电影/游戏IP公仔（含电影名）**

```
[品牌名] [IP角色名] [尺寸] Action Figure from [电影/游戏名], [关节数量]+ Points of Articulation, Officially Licensed, [关键词]
```

**示例（Sonic 3电影版）：**
> JAKKS Pacific Sonic the Hedgehog 5-Inch Shadow Action Figure from Sonic 3 Movie, 20+ Points of Articulation, Officially Licensed, Movie-Authentic Collectible Gift for Kids Ages 3+

### 1.3 标题写作规则

| 规则 | 说明 | 当前遵守 |
|---|---|---|
| 品牌前置 | 前2-3个词必须是品牌名 | ❌ 缺少JAKKS Pacific |
| 核心词在标题前1/3 | IP角色名靠前 | ✅ |
| 英文标题 | 美国市场必须英文 | ❌ 当前为中文 |
| 具体尺寸 | 标明英寸数 | ✅ |
| 授权信息 | 建立正品信任 | ✅ |
| 200字符限制 | 不超200字符防截断 | ✅（但太短，浪费空间） |
| 场景关键词 | gift, collectible, kids | ❌ 缺失 |

## 二、Bullet Points 模板提取

### 2.1 当前Bullet结构（基础版 — 不推荐模仿）

```
Bullet 1: [尺寸] + [产品名]      ← 仅描述
Bullet 2: [参数]                  ← 仅罗列
Bullet 3: [配件]                  ← 仅罗列
Bullet 4: [年龄范围]              ← 重复标题
Bullet 5: [行动号召]              ← 唯一亮点
```

### 2.2 优化Bullet模板（推荐）

**模板：IP授权公仔 5条递进式Bullet**

```
Bullet 1: [核心功能] — [尺寸]+[角色名]+[关节数]+[核心体验]
         格式："[功能标签] — [尺寸] [角色名] figure with [关节数] articulated joints for [体验描述]. [情感化结尾]"

Bullet 2: [配件功能] — [配件描述]+[来源/灵感]+[场景化描述]
         格式："[配件标签] — Includes [配件] inspired by [IP来源]. Set up your own [场景] adventure!"

Bullet 3: [系列收藏] — [系列介绍]+[角色列表]+[暗示收集全套]
         格式："[收藏标签] — Part of the [品牌] [IP名] collection. Collect [角色A], [角色B], [角色C] and more."

Bullet 4: [品质细节] — [涂装]+[做工]+[耐用性]+[双人群]
         格式："[品质标签] — [涂装描述]. [做工优势]. Perfect for [人群A] and [人群B] alike."

Bullet 5: [授权+礼品] — [授权信息]+[场景建议]+[年龄范围]
         格式："[授权标签] — Authentic [IP方] licensed merchandise. Great for [场景A], [场景B], or [场景C] for ages [年龄]+."
```

### 2.3 完整Bullet示例

```
1. POSEABLE PLAY — 4-inch Modern Sonic the Hedgehog figure with 11 articulated joints for dynamic posing and imaginative play. Bring the iconic video game character to life!

2. EXCLUSIVE ACCESSORY — Includes a blue checkpoint accessory inspired by the classic Sonic video game series. Set up your own Green Hill Zone adventure!

3. COLLECT THEM ALL — Part of the JAKKS Pacific Sonic the Hedgehog collection. Collect Sonic, Shadow, Knuckles, Tails, and more to build your ultimate team.

4. PREMIUM DETAIL — Vibrant paint with clean lines and spot-on character accuracy. Stands on its own and holds up to rough play — perfect for kids and collectors.

5. OFFICIALLY LICENSED — Authentic SEGA-licensed merchandise. Makes a great birthday gift, cake topper, or stocking stuffer for Sonic fans ages 3 and up.
```

## 三、A+ Content 模板提取

### 3.1 品类现状

当前产品及所有竞品均无A+内容。以下为建议模板：

### 3.2 通用A+模板（IP授权公仔）

```
模块1：IP介绍（2-3行）
[角色名] — the world's fastest [角色描述]!
Since [IP首次出现年份], [IP名] has been [影响力描述].

模块2：产品展示（3-4行）
This [尺寸] figure features [关节数] points of articulation for [功能描述].
[涂装细节描述].
Includes [配件列表] for [场景化用途].

模块3：角色系列（2-3行）
Collect the entire [系列名] series:
[角色1] — [一句话描述]
[角色2] — [一句话描述]
[角色3] — [一句话描述]

模块4：品质保证（2-3行）
Officially licensed by [IP方].
Made with [材质], designed for [年龄] and up.
[安全认证信息].

模块5：品牌介绍（1-2行)
Part of the [品牌名] collection.
Visit our store to discover more [IP名] products.
```

### 3.3 A+写作规则

| 规则 | 说明 |
|---|---|
| 角色世界观开场 | 比直接描述产品更有吸引力 |
| 展示具体细节 | "11 articulated joints" 比 "fully poseable" 更具体 |
| 暗示系列购买 | "Collect them all" 驱动复购 |
| 授权信息突出 | "Officially licensed by SEGA" 建立正品信任 |
| 500-800字 | 太长没人看，太短没说服力 |
