# 可复用 Listing 模板 — 建筑微缩积木（非 IP）

> 基于竞品分析提取的最佳实践模板，适用于建筑主题微缩积木、街景模型、MOC 积木品类

## 模板适用范围

- 建筑/街景微缩积木套装
- 非IP积木（原创设计或通用建筑主题）
- 中国 OEM 贴牌积木产品
- 带灯光/特殊功能的积木套装
- LEGO 兼容积木
- STEM 教育定位的积木玩具


## 模板二：Bullet Points 模板

### 建筑积木 Bullet 最佳结构（5条）

```
Bullet 1: 核心产品描述
  [EMOJI]【[Product Name]】[What it is] + [Core feature: floors, rooms, layout].
  [Detailed description of each floor and functional area].

Bullet 2: 特殊功能（灯光/互动）
  [EMOJI]【[Feature Name]】[Special function description].
  [How to use it] + [What makes it special].

Bullet 3: 尺寸 + 展示
  [EMOJI]【Display & Decor】[Exact dimensions H x W x L inches].
  [Where to display] + [Aesthetic description].

Bullet 4: 目标人群 + 礼品
  [EMOJI]【Perfect Gift for [Audience]】[Age range] + [Occasions list].
  [Why it's a good gift].

Bullet 5: 材质 + 拼装体验
  [EMOJI]【Quality & Experience】[Material] + [Building experience].
  [Solo or group play] + [Educational value].
```

### 竞品最佳实践 — NEWABWN Bullet 分析

| 优势 | 可复制模式 |
|------|-----------|
| ☕️ Emoji 每条统一 | 品类识别度高，提升可读性 |
| 功能递进（结构→灯光→展示→礼品→品质） | 逻辑清晰，信息密度高 |
| 场景化描述（咖啡吧、沙发区、露台） | 帮助买家想象使用场景 |
| 每条 200-300 字符 | 接近 500 字符上限的一半，可再优化 |

### 关键 Bullet 要素清单（建筑积木品类）

- [ ] 楼层/房间布局描述（每层有什么）
- [ ] 特殊功能（灯光、可动部件、隐藏空间）
- [ ] 精确尺寸（H × W × L 英寸）
- [ ] 兼容性声明（Compatible with major building block brands）
- [ ] 目标年龄
- [ ] 礼品场景（Christmas, Birthday, Valentine's 等）
- [ ] 材质（ABS plastic / Non-toxic）
- [ ] 拼装难度/时间
- [ ] 说明书质量
- [ ] 配件列表（含什么/不含什么，如电池）

## 模板三：A+ Content 模板

### 建筑积木 A+ 标准结构

| 模块 | 内容 | NEWABWN 实际使用 |
|------|------|-----------------|
| 品牌横幅 | Logo + 品牌名 | ❌ 缺失 |
| 产品标题 | 大图 + 简短描述 | ✅ 有 |
| 功能展示 | 灯光开/关对比图 | ✅ 有 |
| 拼装体验 | 说明书/拼装过程 | ✅ 有 |
| 教育价值 | STEM / 创造力 / 逻辑思维 | ✅ 有 |
| 礼品场景 | 多场合送礼建议 | ✅ 有 |
| 尺寸展示 | 与常见物品对比 | ❌ 缺失 |
| 系列推荐 | 同系列其他产品 | ⚠️ 仅 1 款 |

### 建议的 A+ 优化

| 优化项 | 内容 | 优先级 |
|--------|------|--------|
| 添加尺寸参照图 | 成人手 / minifig / 信用卡 | 🔴 P0 |
| 添加拼装完成场景图 | 书桌/书架/办公桌展示 | 🟡 P1 |
| 添加品牌故事 | "我们是谁 / 为什么做建筑积木" | 🟡 P2 |
| 扩展系列推荐 | 3-5 款同系列建筑（花店/书店/餐厅） | 🟡 P2 |

## 模板四：Review 管理模板

### 建筑积木常见差评及回复

| 差评类型 | 典型评论 | 建议回复要点 |
|----------|---------|-------------|
| 尺寸不符预期 | "This thing is TINY" | 道歉 + 提供精确尺寸 + 建议查看图片参照 |
| 缺失零件 | "missing pieces" | 提供补件服务 + 邮寄地址收集 |
| 拼装困难 | "pieces harder to snap" | 解释兼容积木的公差特点 + 质量改进承诺 |
| 结构不稳 | "flimsy" | 建议展示时避免频繁移动 + 加固建议 |
| 包装损坏 | "box destroyed" | 道歉 + 重新发货或退款 |
