## 一、标题模板提取

### 1.1 当前标题结构（Mattel标准格式）

```
[IP全名] + [系列名] + [定位] + [角色名] + [子系列] + [尺寸] + [年代/来源] + [角色描述] + [品质关键词]
```

实际拆解：
- Masters of The Universe Origins → IP全名+系列名
- 豪华(Deluxe) → 定位
- Ram Man → 角色名
- 200X 卡通系列 → 子系列
- 5.5 英寸 → 尺寸
- 1980 年代电视 → 年代/来源
- 人类击打英雄 → 角色描述
- 细节设计 → 品质关键词

### 1.2 通用标题模板（收藏品IP公仔）

**模板A：收藏品Deluxe公仔**

```
[品牌] [IP全名] [系列名] Deluxe [角色名] [子系列], [尺寸]-Inch [年代/来源] [角色描述], [关键参数], [配件], [收藏/礼品关键词]
```

**示例（通用）：**
> Mattel [IP Name] Origins Deluxe [Character] 200X Cartoon Collection, 5.5-Inch Retro 80s TV Action Figure, 16 Points of Articulation, [Accessory] & Collector Card, Nostalgic Collectible Toy Gift

### 1.3 标题写作规则

| 规则 | 说明 | Mattel遵守 |
|---|---|---|
| 品牌前置 | 前2-3个词 | ❌ 缺少"Mattel" |
| IP全名在前 | 完整IP名在标题开头 | ✅ |
| 角色名突出 | 核心角色名清晰 | ✅ |
| 具体尺寸 | 标明英寸数 | ✅ |
| 子系列区分 | 200X vs 原始系列 | ✅ |
| 参数关键词 | 关节数、配件 | ❌ 缺少 |
| 收藏关键词 | collectible/display | ❌ 缺少 |

## 二、Bullet Points 模板提取

### 2.1 Mattel的"剧情驱动式"Bullet结构

```
Bullet 1: IP怀旧开场 → "回传200X玩具系列的美好回忆"（建立情感连接）
Bullet 2: 角色专属剧情 → "灵感来自'海妖之歌'情节"（深度粉丝专属内容）
Bullet 3: 产品规格 → "5.5英寸+配件描述"（实用信息）
Bullet 4: 现代化升级 → "2000年代玩具的现代化+16关节"（收藏价值）
Bullet 5: 包装/收藏 → "收藏卡+窗口盒子"（收藏者关注点）
Bullet 6: 年龄/人群 → "适合6岁以上MOTU粉丝"（购买决策）
```

### 2.2 通用Bullet模板（收藏品IP公仔）

**模板：6条剧情驱动式Bullet**

```
Bullet 1: [IP怀旧] — [系列名] [角色名] figure brings back fond memories of [IP来源/年代].
         格式："[系列定位] [角色名]公仔将带回[年代/IP来源]的美好回忆。"

Bullet 2: [角色剧情] — Inspired by [具体剧集/情节]. [角色动作描述].
         格式："这款人偶的灵感来自[情节名]。[角色在情节中的关键行动]。"

Bullet 3: [产品规格] — [尺寸] figure with [配件描述], taken directly from packaging.
         格式："以[尺寸]设计,这款[角色描述]的动作公仔从包装中取出,手持[配件]。"

Bullet 4: [现代化升级] — Modern update of [原始版本], featuring [关节数] points of articulation.
         格式："这款[角色]是[年代]玩具的现代化产品。[关节数]个关节点可以摆出[姿势描述]。"

Bullet 5: [包装/收藏] — Includes [收藏品] + [包装描述].
         格式："包含[收藏品],人物装在带有[包装特色]的[盒子类型]中。"

Bullet 6: [年龄/人群] — Suitable for [年龄] and up. For [IP] fans.
         格式："这款[系列名][角色]适合[年龄]及以上的[IP]粉丝。"
```

### 2.3 Bullet写作规则

| 规则 | 说明 | Mattel遵守 |
|---|---|---|
| 剧情专属内容 | 每个角色Bullet 2提及不同剧集 | ✅ 最佳实践 |
| 关节参数 | 明确标注关节数 | ✅ |
| 配件具体描述 | 列出每个配件名称 | ✅ |
| 包装信息 | 窗口盒/豪华图形 | ✅ |
| 年龄明确 | 6岁+ | ✅ |
| 英文语言 | 美国市场需英文 | ❌ 全中文 |
