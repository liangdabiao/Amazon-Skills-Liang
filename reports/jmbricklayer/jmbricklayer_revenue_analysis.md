## 1. 销售估算方法论

### 1.1 核心估算逻辑

本报告基于**评论数驱动法**估算月销量，这是亚马逊第三方卖家分析中最广泛使用的方法论之一。其核心原理是：评论数与历史累计销量之间存在稳定的比例关系。

### 1.2 三大估算方法

#### 方法一: 评论/销量比法 (Review-to-Sales Ratio)

| 指标 | 说明 |
|------|------|
| **行业标准比** | 每1条评论 ≈ 20-50件销量 |
| **积木品类特征** | 积木玩具评论意愿较高，约每1条评论 ≈ 15-30件销量 |
| **JMBricklayer修正** | 中国品牌、价格带偏低，评论率偏低，取 **1:20-1:35** 区间 |

```
月销量估算 = (总评论数 / 上架月数) × 评论/销量比 × 月数
```

#### 方法二: 评论加速度法 (Review Velocity)

通过分析评论积累速度推算当前销售活跃度:

- **活跃期产品** (日均新增 >1条评论): 1评论 ≈ 15-20件销量
- **稳定期产品** (日均新增 0.3-1条评论): 1评论 ≈ 25-35件销量
- **衰退期产品** (日均新增 <0.3条评论): 1评论 ≈ 35-50件销量

#### 方法三: 每日评论推算法

```
日均评论数 = 总评论数 / 上架天数
月销量 = 日均评论数 × 30 × 评论/销量比
```

### 1.3 估算参数设定

| 参数 | 保守值 | 中性值 | 乐观值 |
|------|--------|--------|--------|
| 评论/销量比 | 1:35 | 1:25 | 1:20 |
| 评论加速度因子 | 0.6x | 1.0x | 1.4x |
| 平季月系数 | 0.8x | 1.0x | 1.2x |

### 1.4 上架时间推断

根据评论数量和品类特征推断产品上架时间:

| 评论数区间 | 推断上架时间 | 依据 |
|------------|-------------|------|
| 1000+ | 18-24个月 | 积木品类月均评论约40-60条 |
| 300-999 | 8-18个月 | 中等积累速度 |
| 100-299 | 4-10个月 | 新品期过后的成长阶段 |
| 50-99 | 2-5个月 | 新品成长期 |
| <50 | <3个月 | 新品上市初期 |
