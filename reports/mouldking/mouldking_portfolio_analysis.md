## 一、店铺概览

### 1.1 品牌定位

Mould King（MK）定位为**中高端兼容积木品牌**，核心聚焦于大颗粒/技术类积木套装，产品线覆盖工程机械、车辆模型、动力引擎、火车、影视IP（海盗船、Star Wars风格武器）、游乐园设施及创意摆件等多个垂直品类。品牌以**高零件数、高还原度、高性价比**为竞争卖点，直接对标LEGO Technic/Creator Expert等高端产品线，但价格通常低30%-60%。

## 九、综合诊断与建议

### 9.1 优势

1. **引擎品类壁垒极高**：V8引擎系列累计评论超3000条，是该品类的绝对领导者
2. **价格带覆盖全面**：从$25.99到$450.00，覆盖各消费层级
3. **产品迭代速度快**：持续推出升级版（如13112升级版）和扩展系列（光剑5款+）
4. **IP拓展能力强**：从Star Wars光剑到海贼王武士刀，快速抓住热门IP

### 9.2 风险与问题

1. **Railcar系列品控风险**：12040意大利马可尼(2.1分)和12045火车(2.7分)严重拖低整体评分，需紧急排查
2. **大量"未标价"产品**（~37%）：可能存在供应链/库存/ listing维护问题
3. **同款多ASIN**现象：分散评论权重，削弱单一listing的排名和可信度
4. **Street View品类尚未成熟**：飞行旋转仅2.9分、过山车有3.4分的低分版本，品控不稳定
5. **8506件超大型起重机(17007/17008)**评分仅3.5-3.7，作为品牌旗舰产品表现不佳

### 9.3 战略建议

| 方向 | 建议 |
|------|------|
| 短期 | 修复12040/12045/17008等低分产品，排查品控或说明书问题；清理重复ASIN，合并同款评论 |
| 中期 | 持续扩展光剑+武士刀的Star系列，打造"积木武器"第一品牌心智；加大植物/动物生活品类投入 |
| 长期 | 稳固引擎系列护城河，探索电机化/智能化升级；强化火车系列口碑，打造"积木火车专家"定位 |
